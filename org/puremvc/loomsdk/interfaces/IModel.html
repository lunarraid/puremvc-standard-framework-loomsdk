IModel

public interface
org.puremvc.loomsdk.interfaces.IModel

Inheritance

Description

The interface definition for a PureMVC Model.

In PureMVC, IModel implementors provide access to IProxy objects by named lookup.

An IModel assumes these responsibilities:

Maintain a cache of IProxy instances Provide methods for registering, retrieving, and removing IProxy instances
